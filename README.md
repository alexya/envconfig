# vimconfig
Set up vim configurations quickly on Linux or Mac OS X
* customized .vimrc
* plug-in management tool - pathogen
* some common useful plug-ins

# Installation
    cd ~/
    git clone https://github.com/alexya/vimconfig.git
    cd vimconfig
    bash setupvim.sh
